# БЫСТРОЕ РЕШЕНИЕ ПРОБЛЕМЫ АВТОРИЗАЦИИ

## Проблема
Логин не работает: "Invalid email or password" (401)

## Решение: Создайте тестового пользователя через API

### Шаг 1: Откройте новый терминал

### Шаг 2: Создайте тестового админа
```bash
curl -X POST http://localhost:5000/api/test/create-test-user
```

Это создаст пользователя:
- **Email:** `test@wfa.com`
- **Пароль:** `testpass123`
- **Роль:** Admin ⭐

### Шаг 3: Войдите на сайте
1. Перейдите на http://localhost:5173/login
2. Введите:
   - Email: `test@wfa.com`
   - Пароль: `testpass123`
3. Нажмите "Войти"

### Шаг 4: Проверьте что вы админ
После входа нажмите на ссылку "Админ" в навигации - должна открыться админ-панель!

---

## Другие полезные команды

### Посмотреть всех пользователей
```bash
curl http://localhost:5000/api/test/list-users
```

### Удалить всех пользователей
```bash
curl -X DELETE http://localhost:5000/api/test/delete-all-users
```

---

## Почему это работает?

Тестовый endpoint создает пользователя напрямую в базе с правильным bcrypt хешем пароля, так что вы точно знаете email и пароль для входа.

**ВАЖНО:** Эти тестовые endpoints должны быть удалены перед деплоем в продакшн!
